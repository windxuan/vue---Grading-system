问卷页思路(answer)
  1. 这是一个问卷调查系统(第一级标签)--'问卷调查系统'
  2. 这套问卷的目的是帮助自己把握在团队中的定位(灰色小字)--'团队角色认知'
  3. 进入该页面,展示给用户的就是问卷列表(第二级标签)--'问卷列表'
  4. 选择问卷列表后才是答题界面
  5. 答题界面与个人汇总放在同一级, 方便用户进行跳转 单用户未答题时,个人汇总为空


Bug1: 一开始将获取问卷url放在列表页选中列表某一行时，在选中列表页选中某行数据,生成url,提交给vuex, 答题页面导航栏解封;
  此时回到列表页面, 点击某一行,再次生成url,覆盖原vuex, 如果此次点击与上次点击选中问卷不相同,就会出现答题完提交时,
  答题题目与列表url不一致的情况,导致出错。
